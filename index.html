Bem-vindos ao CodeCollab!
Esta é a homepage do nosso projeto bimestral de Aplicação a Web-1. Esta página, além do conteúdo visto, apresenta algumas outras tecnologias que são muito úteis para o Desenvolvimento Web.

Sobre o Projeto

Este projeto é algo essencial para colocar tecnologias essenciais para o Desenvolvimento Web em prática, podendo assim desenvolver uma visão lógica e prática sobre. A aplicação, de forma geral, utiliza HTML5 e CSS3 para estruturar e estilizar páginas de maneira moderna e responsiva. Além disso, aproveitou-se do conhecimento já adquirido no mundo do versionamento de código, utilizando Git e GitHub, garantindo que todas as mudanças e colaborações fossem organizadas, documentadas e seguras.

Nesse projeto, utilizamos as seguintes tecnologias:

  • HTML5: A base de tudo. O conteúdo foi estruturado com as tags semânticas, garantindo melhor acessibilidade e otimização para mecanismos de busca.
  • CSS3: As páginas obtiveram estilos incríveis, utilizando flexbox, grid e animações. A responsividade foi uma prioridade!
  • Git & GitHub: Todo o código foi versionado, permitindo a colaboração em equipe e mantendo um histórico claro de todas as mudanças. Se algo deu errado, sempre foi possível voltar atrás!
  • VS Code: Sem dúvidas, o melhor editor de código. Com suas extensões e funcionalidades, o desenvolvimento foi muito mais fluido e produtivo.
