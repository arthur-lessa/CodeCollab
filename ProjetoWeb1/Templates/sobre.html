Sobre o Projeto:


Tecnologias Utilizadas

Desenvolvimento com HTML5

HTML5: O responsável pela estruturação do nosso projeto. Utilizamos tags semânticas como <nav>, <section>, e <footer> para garantir que nosso código fosse bem organizado e acessível.

Desenvolvimento com CSS3

CSS3: Estilizando as páginas. Utilizamos Flexbox e Grid Layout para criar layouts responsivos, e animações em CSS para proporcionar uma experiência visualmente melhor ao usuário.

Versionamento de Código com Git & GitHub

Git: Controle de Versão Eficiente. Com o versionamento de código, pudemos trabalhar de forma organizada e segura, registrando cada mudança com commits claros.

GitHub: Colaboração e Publicação. GitHub serviu como nosso repositório remoto, facilitando a colaboração e a revisão de código através de commits, merges e pushs.

Ferramentas de Desenvolvimento

VS Code: O Visual Studio Code se destacou como a escolha principal de ambiente de desenvolvimento. Com plugins eficientes para HTML, CSS e Git, o gerenciamento de código foi simplificado, e a produtividade foi otimizada grandemente. Sua interface intuitiva e recursos robustos tornaram o desenvolvimento mais ágil e conciso, garantindo projetos de alta qualidade de forma rápida.

Fluxo de Trabalho do Projeto

1: Planejamento e Estruturação

No início do projeto, realizamos um planejamento detalhado, definindo o escopo, objetivos e requisitos. Organizamos a estrutura de pastas e arquivos para garantir uma base sólida para o desenvolvimento. Além disso, organizamos quanto tempo e qual a disponibilidade da equipe para lidar com o projeto.

2: Desenvolvimento

Iniciamos o desenvolvimento criando o HTML básico que define a estrutura do projeto, a HomePage, e logo em seguida, as outras páginas (Sobre e Desenvolvedores). Em paralelo, estilizamos os layouts utilizando CSS para garantir uma apresentação visual atraente e funcional.

3: Versionamento com Git/Github

Utilizamos o Git e o GitHub para controlar as versões e alterações do projeto. Criamos repositórios locais e remotos para gerenciar alterações, realizando sempre revisões de código, feedbacks e integrações contínuas para melhorar o projeto. Essas ferramentas foram muito úteis para o nosso projeto, pois facilitaram o trabalho em equipe e garantiram a integridade do código ao longo de seu desenvolvimento.

4: Desenvolvimento no VS Code

O Visual Studio Code foi a nossa principal ferramenta de desenvolvimento. Utilizamos seus recursos avançados, como extensões específicas para HTML, CSS, para aumentar a produtividade e qualidade do código. Utilizamos também sua ótima estruturação de arquivos para melhor organização e orientação.

5: Testes e Ajustes Finais

Realizamos testes complexos para verificar a funcionalidade e usabilidade do projeto. Corrigimos possíveis bugs e realizamos ajustes finais no design e no código para garantir uma experiência de usuário consistente e satisfatória.

6: Deploys e Publicação

Finalmente, preparamos o projeto para o lançamento. Utilizamos serviços como GitHub Pages para publicar o projeto online, garantindo que ele esteja acessível na web.
